<Ctrl-k><characters> = insert digraphs (æ,π...) [Insert Mode]
<Ctrl-k>*p = insert π [Insert Mode]
<Ctrl-k>Eu = insert € [Insert Mode]
<Ctrl-k>+- = insert ± [Insert Mode]
<Ctrl-k>OK = insert ✓ [Insert Mode]
<Ctrl-k>XX = insert ✗ [Insert Mode]
<Ctrl-k>AN = insert ∧ ('and' symbol) [Insert Mode]
<Ctrl-k>OR = insert ∨ ('or' symbol) [Insert Mode]
<Ctrl-k>(- = insert ∈ [Insert Mode]
<Ctrl-a> = insert the last content inserted [Insert Mode]
<Ctrl-@> = insert the last content insert and quit insert mode [Insert Mode]
<Ctrl-h> = delete character before the cursor [Insert Mode]
<Ctrl-l> = delete  character after the cursor (like Del) [Insert Mode]
<Ctrl-w> = delete word under cursor [Insert Mode]
<Ctrl-o> = execute one normal or command-mode and return to insert [Insert Mode]
o = move cursor to opposite side of selection [Visual Mode]
U = Uppercase the selection [Visual Mode]
<Ctrl-v> = Change to visual-bloc mode [Visual Mode]
<Shift-v> = Change to visual mode [Visual-bloc Mode]
